Form validation - how to

What this is

This is code to implement form validation in the kit, with examples of different types of validation you can do - the validation code can be copied and used in your prototype.

What this isn't

This isn't a guide as to when to use each type of component, and not what the error messages should be. For your protoype you should get the latest guidance and component HTML/Nunjucks macros from the design system
